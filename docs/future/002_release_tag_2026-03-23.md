# 002 - Tag de Release v0.3.0-baseline

**Data:** 2026-03-23
**Projeto:** drm_transformer
**Status:** pendente (depende de baseline treinado + resultados)

## Objetivo

Criar release tag `v0.3.0-baseline` com artefatos reproduziveis apos
completar treino do baseline e ablacoes.

## Pre-requisitos

- [ ] Dataset CulturaX 20B tokens baixado e processado
- [ ] Baseline small_1m treinado com resultados em metrics.json
- [ ] Ablacoes rodadas com results_ablations.md gerado
- [ ] MODEL_CARD.md preenchido com numeros reais
- [ ] Smoke tests passando (8/8)

## Artefatos da Release

- `checkpoints/baseline_1m/final.pt` — checkpoint final
- `checkpoints/baseline_1m/best.pt` — melhor val_loss
- `checkpoints/baseline_1m/metrics.json` — sumario de metricas
- `checkpoints/baseline_1m/run_manifest.json` — ambiente/config
- `results_ablations.md` — tabela comparativa de ablacoes
- `repro_report.json` — relatorio de reproducao

## Procedimento

```bash
# 1. Verificar smoke tests
pytest tests/test_smoke.py -v

# 2. Rodar reproducao completa
python scripts/repro_baseline.py

# 3. Preencher MODEL_CARD.md com numeros
# 4. Commit final

# 5. Criar tag
git tag -a v0.3.0-baseline -m "Baseline reproduzivel: small_1m + ablacoes"
git push origin v0.3.0-baseline

# 6. Criar release no GitHub
gh release create v0.3.0-baseline \
    checkpoints/baseline_1m/final.pt \
    checkpoints/baseline_1m/metrics.json \
    results_ablations.md \
    --title "v0.3.0-baseline: Baseline Reproduzivel" \
    --notes-file docs/future/002_release_notes.md
```
